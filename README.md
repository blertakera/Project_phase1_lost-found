# Project Details

## Project Title: Lost & Found System

## Problem Statement:
Many people lose personal belongings in public places such as universities, parks, or public transport. Currently, there is no simple and efficient digital system where users can report or search for lost and found items.

## Proposed Solution:
We propose a digital platform where users can report lost or found items, upload photos, and search through listings using filters such as location, category, and keywords. The system will also allow communication between users to help return items.

## Project Scope:

### Aim:
To develop a digital platform that simplifies the process of reporting and recovering lost items.

### Objectives:
1. Develop a working prototype of the system  
2. Allow users to report lost items  
3. Allow users to report found items  
4. Implement search filters (location, category, keywords)  
5. Enable communication between users  


## Application Description:
The application will be a web or mobile platform where users can register, post lost or found items, and search for matches. Each post will include details such as description, location, and photos. Users will be able to contact each other through the system, and an admin will manage and moderate content.
